# Typography

The typography using inside Venice and their CSS Variables.

<Typeset
  fontSizes={['12px', '14px', '16px', '24px', '38px']}
  sampleText="Venice is the best!"
/>

## Using

### Variable values

| Variable             | Value                 |
| -------------------- | ----------------------|
| `var(--font-family)` | 'Roboto', sans-serif' |

### Class values

| Typography           | Value                 |
| -------------------- | ----------------------|
| Panel                | `jsm-t-panel`         |
| Title                | `jsm-t-title`         |
| Subheader            | `jsm-t-subheader`     |
| Body                 | `jsm-t-body`          |
| Body2                | `jsm-t-body2`         |
| Caption              | `jsm-t-caption`       |
| Button               | `jsm-t-button`        |

We have variables to font and letter spacing, you just need to use on `font` and `letter-spacing` property. You can just use a class to add a typography styles too.

<Preview style={{
  padding: `var(--space-xs) var(--space-sm)`
}}>
  <div style={{
    font: `var(--typography-panel-font)`,
    letterSpacing: `var(--typography-panel-letter)`
  }}>Panel</div>
</Preview>

```html
<h1 class="jsm-t-panel">Panel</h1>
```

```css
foo {
  font: var(--typography-panel-font);
  letter-spacing: var(--typography-panel-letter);
}
```
